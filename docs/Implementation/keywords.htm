Reserved Keywords

bold( )
Embeds the argument in HTML <b> tags.
cell( rowNumber, colNumber, spreadsheet )
Returns a reference to the HTML table cell element referenced by logical row and column numbers. If spreadsheet is omitted, the state.currentSpreadsheet is used. If rowNumber is a string, a named cell is retrieved and colNumber is ignored.
col
Column number of the currently evaluated cell.
color(fgc, bgc, expression)
Embeds the expression argument in HTML <span> tags and sets foreground and background color styles.
currentCell
A reference to the currently evaluated cell.
cv( rowNumber, colNumber, spreadsheet )
A shortcut for cell( row, col ).spreadsheetCore.value. Returns the cell value (cell.spreadsheetCore.value).
italic( )
Embeds the argument in HTML <i> tags.
row
Row number of the currently evaluated cell.
rr( rowOffset )
Returns a reference to a row at the specified distance from the current row.
rc( columnOffset )
Returns a reference to a column at the specified distance from the current column.
underline( )
Embeds the argument in HTML <u> tags.
