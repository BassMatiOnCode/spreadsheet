Requirements

Formulas must be able to reference arbitrary cells.
The library will implement a cell function.
Rows and columns must be referenced by absolute coordinates.
The cell function accepts numerical parameters that are considered to be absolute row and column indices.
Rows and columns must be referenced by relative coordinates.
The rr (row relative) and cr (column relative) functions accept a numerical argument that is interpreted as offset to the current cell coordinates.
The user may insert and delete rows and columns.
The library will provide functions for adding and deleting rows and columns (addRow, deleteRow, addColumn, deleteColumn).
If columns or rows are inserted or deleted, cell references must be updated to keep formulas intact.
The library will implement the necessary functionality in the addRow, deleteRow, addColumn and deleteColumn functions.
The user must be able to copy and paste rows and columns.
The library will provide cutRow, copyRow, pasteRow, cutColumn, copyColumnand pasteColumn functions.
